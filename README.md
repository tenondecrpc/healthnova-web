# HealthNova Web

Web dashboard for **HealthNova**, a preventive health platform that helps users understand and act on their health data through personalized insights, trend analysis, and actionable recommendations.

The application consumes a separate backend API authenticated via AWS Cognito, presenting health information in a secure, accessible, and privacy-first interface.
